# 홈택스 자동화 스크립트 실행 가이드

## 방법 1: Playwright 자동화 (권장)
1. 설치 (이미 완료됨):
   npm install playwright
   npx playwright install

2. 실행:
   node hometax_automation.js

## 방법 2: 브라우저 콘솔에서 직접 실행 (가장 확실)
1. 홈택스 세금계산서 발행 페이지에 접속
2. F12 키를 눌러 개발자 도구 열기
3. Console 탭 선택
4. simple_automation.js 파일 내용을 복사해서 붙여넣기 후 Enter

## 방법 3: 북마클릿 사용
1. bookmarklet.js 파일 내용을 복사
2. 브라우저 북마크로 추가 (URL 부분에 붙여넣기)
3. 홈택스 페이지에서 해당 북마크 클릭

## 기능
- 사업자등록번호 "5148700192" 자동 입력
- 확인 버튼 자동 클릭  
- Alert 창 확인 버튼 자동 클릭

## 추천 방법
방법 2 (콘솔 실행)가 가장 확실하고 안전합니다.
홈택스 페이지에서 F12 > Console > simple_automation.js 내용 붙여넣기
